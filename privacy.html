Política de privacidad

Última actualización: mayo 2025 · vpo.jjalcantara.dev

Responsable

Jesús Jiménez Alcántara — jjalcantara.dev
Contacto: contact@jjalcantara.dev

Esta herramienta se ofrece sin ánimo de lucro como análisis público de documentación disponible sobre la promoción Galivivienda Fortalezas de Vélez. No recoge, almacena ni trata datos personales identificables de sus visitantes.

La información recogida en esta herramienta puede contener inexactitudes o estar desactualizada. Se irá corrigiendo y mejorando conforme se disponga de más datos contrastados. Si detectas algún error, escríbeme a contact@jjalcantara.dev.

Datos que esta web NO recoge

Esta web no tiene analíticas, no usa cookies de seguimiento, no muestra publicidad y no transmite ningún dato tuyo a terceros más allá de lo estrictamente necesario para servir la página. Tampoco guarda formularios, registros de usuario ni ningún tipo de perfil.

Servicios de terceros

Al cargar la página, el navegador establece conexiones con dos servicios externos que pueden procesar tu dirección IP:

Vercel Inc.
Proveedor de alojamiento web. Sirve los archivos estáticos de esta página desde sus servidores. Como cualquier proveedor de hosting, Vercel puede registrar metadatos de las peticiones HTTP (dirección IP, fecha, navegador, URL solicitada) con fines de seguridad y operativos.
Política de privacidad: vercel.com/legal/privacy-policy ↗
Google LLC — Google Fonts
Las tipografías (Inter, DM Mono) se cargan desde servidores de Google Fonts. Esta petición puede incluir tu dirección IP y datos técnicos del navegador. Google puede registrar estos datos conforme a su política de privacidad.
Política de privacidad: policies.google.com/privacy ↗

Almacenamiento local (localStorage)

Esta web guarda en el almacenamiento local de tu navegador (localStorage) una única clave — cookie-ok — para recordar que has cerrado el aviso de privacidad. No es una cookie técnicamente, no tiene fecha de expiración del servidor y no se transmite a ningún servidor. Puedes borrarlo en cualquier momento desde los ajustes de tu navegador (DevTools → Application → Local Storage).

Tus derechos (RGPD)

En la medida en que esta web procese algún dato personal, tienes derecho a acceder, rectificar, suprimir, oponerte y solicitar la portabilidad de tus datos. Dado que no almacenamos datos identificables, el ejercicio práctico de estos derechos aplica principalmente a los servicios de terceros mencionados arriba, contactando directamente con Vercel o Google.

Para cualquier consulta sobre esta política escríbeme a contact@jjalcantara.dev. Tienes también derecho a presentar una reclamación ante la Agencia Española de Protección de Datos (aepd.es) ↗.
